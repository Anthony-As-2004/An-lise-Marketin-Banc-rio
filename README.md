# Análise de Eficiência em Campanhas de Marketing Bancário

Este projeto compõe meu portfólio de Análise de Dados e foi desenvolvido integralmente no Microsoft Excel. O estudo utiliza o dataset Bank Marketing para avaliar o desempenho de campanhas de telemarketing ativo focadas na contratação de depósitos a prazo.

## Objetivo
Analisar o histórico das campanhas para identificar padrões demográficos e financeiros dos clientes, otimizando o direcionamento de futuras abordagens comerciais e a alocação de recursos da equipe de vendas.

## Ferramentas Utilizadas
* Microsoft Excel
* Power Query (Importação e tratamento de delimitadores)
* Tabelas Dinâmicas (Agregação de dados e cálculo de taxas de conversão)
* Histogramas (Modelagem exploratória)

## Principais Insights

### 1. Conversão por Perfil Demográfico
A análise de conversão revelou uma disparidade entre o esforço operacional e a eficiência de vendas. Enquanto a grande maioria das ligações é feita para os setores administrativo e operacional (que convertem entre 10% e 11%), os segmentos com maior taxa de sucesso na base são os Estudantes (31,43%) e Aposentados (25,23%).

### 2. Eficiência por Canal
O telefone celular é o canal de comunicação mais utilizado (63,47% do volume total) e o que apresenta a maior eficiência para a conversão final de contratos.<br>

## Conclusão
Os dados sugerem que a estratégia da campanha deve ser realinhada para priorizar estudantes e aposentados, nichos que demonstram maior propensão de compra. Adicionalmente, recomenda-se evitar a oferta de depósitos a prazo para clientes com empréstimos ativos ou histórico de inadimplência, uma vez que as análises comprovam uma liquidez reduzida nestes perfis.
